# Hướng dẫn cài đặt và sử dụng

## Cài đặt
1. Clone repository này về máy của bạn.
2. Chạy lệnh `npm install` để cài đặt các thư viện cần thiết.
3. Biên dịch smart contract sử dụng Remix hoặc Truffle.
4. Chạy server bằng lệnh `npm start`.

## Sử dụng
- Mở trình duyệt và truy cập `http://localhost:3000` để sử dụng ứng dụng.
